## 📌 Sobre o projeto

Este projeto foi desenvolvido com HTML, CSS e JavaScript com o objetivo de praticar lógica de programação, manipulação de dados e interação com o usuário.

A aplicação permite adicionar números a uma lista e realizar análises sobre os dados informados, como quantidade, maior valor, menor valor, soma e média.

## 🚀 Tecnologias utilizadas

* HTML (estrutura da página)
* CSS (estilização e layout)
* JavaScript (lógica, manipulação do DOM e tratamento de dados)

## 🎯 Funcionalidades

* Adição de números a uma lista
* Validação de valores (permitindo apenas números entre 1 e 100)
* Prevenção de valores duplicados
* Análise dos dados inseridos:

  * Quantidade de números
  * Maior valor
  * Menor valor
  * Soma total
  * Média dos valores
* Exibição dinâmica dos resultados
